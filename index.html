Endorphin Browser

Endorphin is a cross platform web browser built using Qt and WebKit.

It is a fork of the Arora browser. Arora was created by Benjamin C. Meyer, Endorphin was created by Aaron Dewes.

The goal of Endorphin is to create a lightwight, fast and simple browser, that has many features and is compatible with most websites. Endorphin is still in development, so no releases are available now, but the source code is available on GitHub.

Default version

This version is recommended for most users, it renders websites better than the WebKit version, and has a better Web Inspector, but might use more RAM.

WebKit version

This version is recommended for older computers or computers with less available RAM, it is not as good as the default version at rendering, but should still work with most websites.
